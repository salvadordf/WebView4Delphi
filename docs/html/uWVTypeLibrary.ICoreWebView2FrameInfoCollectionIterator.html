function Get_hasCurrent(out hasCurrent: Integer): HResult; stdcall;

`TRUE` when the iterator has not run out of `FrameInfo`s. If the collection over which the iterator is iterating is empty or if the iterator has gone past the end of the collection, then this is `FALSE`.
