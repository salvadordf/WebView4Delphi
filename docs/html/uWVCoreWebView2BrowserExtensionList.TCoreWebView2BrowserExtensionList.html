Declaration

type TCoreWebView2BrowserExtensionList = class(TObject)

Description

Provides a set of properties for managing browser Extension Lists from user profile. This includes the number of browser Extensions in the list, and the ability to get an browser Extension from the list at a particular index.

<see href="https://learn.microsoft.com/en-us/microsoft-edge/webview2/reference/win32/icorewebview2browserextensionlist">See the ICoreWebView2BrowserExtensionList article.)
